1) Se debe clonar el repositorio para el cual se debe ejecutar el siguiente comando:
git clone https://github.com/Noguik06/Tarea6.git nombreDeseado

2) Se debe entrar a la carpeta al nombre del proyecto que le di nombreDeseado

3) Ejecutar la sentencia mvm clean install

4) Ejecutar el comando heroku local web

5) Entrar en la dirección localhost:5000/tarea6 para ver la versión desplegada en un servidor local

6) Entrar en la siguiente dirección para ver la versión de la aplicación en un servidor en la nube:
http://aqueous-brushlands-10523.herokuapp.com/tarea6/
